Projects

After School Fight Club Source code

I made a fictive webshop with a retro look in React that "sells" input devices for fighting games. It uses Firebase's Firestore to store the listed products. And the cart's content is linked to the user's account.

Tommy Hilfiger landing page Source code

Landing page for Tommy Hilfiger made with react-bootstrap.

Netflix Clone / Modal Source code

Together with a team I worked on a Netflix clone. I made the inner contents of the Modal component and attempted to make it looks like the real one as much as possible. In my opinion it is even better than the original because it is more responsive to smaller screens. It was created in React with react-bootstrap and the data comes from TMDB API.

Weather App Source code

I made a weather app in React that fetches live data from the OpenWeatherMap API and background images from the Teleport API.
